# Language Learning Course Blueprint (A0 → B2)

## Philosophy

This course is designed for **fast, practical language acquisition** using:

- Sentences over isolated vocabulary
- Pattern recognition over grammar memorization
- High repetition early, variation later
- Real-world usage as soon as possible

No grammar study is required. Learners absorb structure naturally through exposure.


## Audio & Playback System

Learners must be able to:
- Very slow audio (0.50x)
- Slow audio (0.75x)
- Play at normal speed (1.0x)
- Slightly increase speed (1.25x)
- Repeat sentences easily

This supports:
- pronunciation development
- listening comprehension
- transition to real speech speed


## Shadowing (introduced gradually)

At A1 and beyond:
- learners can repeat sentences aloud
- optional use of slowed audio
- no formal instruction required


# Course Structure

Each level contains:

## 1. Pattern Decks (Core Engine)
- Controlled sentences
- High repetition
- Teach structure implicitly

## 2. Usage Decks (Guided)
- Real-life sentences
- Slight variation
- Reinforce patterns

## 3. Interaction Decks
- Conversations
- Back-and-forth structure
- Speaking practice

## 4. Exposure Decks
- Large input (e.g., Tatoeba, stories)
- Introduced gradually

## 5. Quick Tips Layer (Inline)
- Short explanations (1–2 sentences)
- Triggered at confusion points
- No memorization required

# System Rules

## 1. Pattern Before Exposure

| Level | Exposure |
|------|--------|
| A0 | None |
| A1 | Minimal |
| A2 | Controlled |
| B1 | Moderate |
| B2 | Heavy |

---

## 2. Vocabulary Strategy

- No isolated vocabulary memorization
- Words always appear in sentences
- High repetition early
- Expansion later

---

## 3. Grammar Approach

- Never taught explicitly
- Learned through repeated patterns
- Supported by Quick Tips only

---

## 4. Quick Tips

### Purpose
Reduce confusion without teaching grammar.

### Examples
- Silent letters
- Liaison (linking sounds)
- Elision (j’ai, l’ami)
- Sound vs spelling differences

### Rules
- 1–2 sentences max
- No technical language
- No memorization required

---

## 5. Personalization System

Users can:
- Add sentences from their own life
- Generate translations + audio
- Build custom decks

This reinforces real-world usage and accelerates learning.

---

# Design Principles

- Short sentences
- High repetition
- Minimal vocabulary expansion early
- Real-world usefulness
- Confidence before complexity
