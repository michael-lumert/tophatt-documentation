## Final review findings

### 1. The blueprint should bless small sound/symbol decks

The current A0.3 subdecks range from **15 to 36 cards**, and that appears to be a feature, not a defect. Special sound and special-character decks should no longer be forced into the old 40-card minimum. A better rule:

- **12–20 cards**: micro sound/symbol deck
- **20–40 cards**: focused sound/symbol deck
- **40–70 cards**: normal focused foundation deck
- **70–90 cards**: standard foundation deck
- **90–125 cards**: large bootstrapping deck, allowed only when the goal is broad confidence or high-value article/gender repetition

### 2. The blueprint should prefer short phrases over isolated words

The new A0 decks demonstrate an important improvement: single-word cards are often followed by tiny chunks, such as article+noun, adjective+noun, or two-word phrases. This should become a BasicA0 rule:

> Single-word cards are allowed for sound bootstrapping, but most new memorable words should quickly appear in a tiny phrase.

Good A0 phrase types:

- article + noun: `un café`, `une idée`
- adjective + noun: `bonne idée`, `petit problème`
- tiny function phrase: `à moi`, `ou non`, `ici et là`
- article/gender reveal: `le café`, `une école`

### 3. Repetition is mostly useful, but needs an upper-bound guideline

Repeated words like `café`, `hôtel`, `idée`, `œuf`, and `téléphone` are not random bloat. They are being reused as anchor words across multiple French-specific problems. For example, `hôtel` supports cognates, circumflex, silent h, `l’`, liaison, and gender.

Recommended guideline:

> Reusing an earlier word is encouraged when the learner has a new job to do with it. Avoid repeating the same word in the same role.

A practical soft cap: **6–8 appearances** for an ordinary anchor word, but allow more for a “teaching workhorse” like `hôtel` if every repeat has a distinct purpose.

### 4. The A0.3 organization is now the right model

The split accent/special-character decks should be treated as the new FrenchA0 model:

- `é` as its own deck
- open e: `è` / `ê`
- tiny accent words: `a/à`, `ou/où`, `sur/sûr`
- circumflex vowels
- cedilla
- ligature `œ`

This is better than one large accents deck because learners can repeat the hard sound/symbol without dragging easier ones along.

### 5. A0.6 Gender & Articles is doing useful repetition

A0.6 is large at 84 cards, but the repetition is pedagogically different from A0.3. It teaches the article partner behavior by showing noun, definite form, and indefinite form. That structure should stay.

Guideline update:

> Article/gender decks may use deliberate triplets when the triplet reveals information the learner cannot see from `l’` alone.

Example: `idée`, `l’idée`, `une idée`.

### 6. A few polishing flags remain

- `mon numéro de téléphone est` is an incomplete fragment. Consider changing it to `mon numéro de téléphone` / “my telephone number”, or postpone full phone-number sentences to A1.
- `désoler` is borderline A0. It is useful only as support for `désolé`; otherwise it can move later.
- `Des pâtes, s’il vous plaît.` is long by token count but formulaic and useful; I would keep it if the deck feels good in audio.
- `fr_a0_6_gender_and_articles_little_partner.txt` has ID gaps. That is not a functional problem if IDs only need to be unique, but renumbering would make the file cleaner.

## BasicA0 blueprint changes needed

1. Replace the old minimum deck-size guideline with a microdeck exception.
2. State that phrases beat isolated words for memory, but single words are still allowed for sound orientation.
3. Add a “reuse with a new job” rule.
4. Add a “language-specific modules” model.
5. Separate BasicA0 from FrenchA0:
   - BasicA0 = universal method
   - FrenchA0 = accents, elision, liaison, gender/articles, silent letters, nasal vowels, special French sound gaps
6. Make `word_count` and `sentence_count` diagnostic only, not pass/fail metrics.
7. Add a rule that references to earlier material should be generic learner-facing recall notes, not internal deck names.

## FrenchA0 blueprint changes needed

FrenchA0 should document the current deck family as the recommended structure:

1. Cognates / familiar-looking words
2. Sound gaps
3. Accent and special-character microdecks
4. Elision/apostrophes
5. Liaison/linking
6. Gender/articles with image-supported first encounters

It should also explicitly allow repeated anchor words when those words serve different French-specific teaching jobs.
